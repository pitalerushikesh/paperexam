# MHCET Exam Paper Practice UI

A React + TypeScript web application that simulates a real exam question paper experience.  
Designed to look and behave like an actual MHCET exam paper, allowing students to solve questions one by one with a blank rough/answer sheet.

---

## Features

- Real exam-paper layout
- One question per page
- Marks displayed for each question
- Large blank answer / rough space
- Previous / Next navigation
- Dedicated exam cover page
- Reusable, modular components
- Material UI-based clean academic design
- Responsive layout
- Print-friendly structure (future-ready)

---

## Exam Flow

### Cover Page

- Exam name (MHCET)
- Year
- Subject
- Total Marks
- LearnOpediA logo

### Question Pages

- Only one question visible at a time
- Question number and marks shown clearly
- Blank space for solving
- Navigation using Previous / Next buttons
